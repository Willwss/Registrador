Esta página permite que você insira informações sobre um produto, incluindo nome, preço, produto, CPF e telefone. Depois de inserir o preço, selecione a quantidade de parcelas desejada na lista de opções. Quando estiver pronto para exportar os dados para uma planilha CSV, insira o nome do arquivo CSV no campo apropriado e clique no botão "Exportar para CSV". Os dados serão baixados automaticamente em um arquivo CSV com o nome especificado.
